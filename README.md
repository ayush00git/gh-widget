# GitHub Contributions Widget (gh-widget)

<img src="public/img/image.png" alt="GitHub Widget Preview" width="500">

**gh-widget** is a custom Android home screen widget designed to bring your GitHub contribution graph directly to your mobile device. Built natively for Android using Kotlin, Jetpack Glance, and WorkManager, the widget seamlessly and periodically synchronizes your daily GitHub activity without requiring an open app.

The widget displays the familiar 53x7 contribution matrix, allowing you to track your coding streaks throughout the year at a glance. It faithfully mimics the iconic GitHub contribution visualization by applying varying brightness levels to the data points corresponding to the intensity of your daily commits, pull requests, and code reviews.
